get /meeting
Get the list of handled meetings. (getMeetings)
Meeting endpoint returns information about the meetings that user has been created or shared. The response includes the display name and details about each meeting.

Return type

Meetings

Example data

Content-Type: application/json
""

Produces

This API call produces the following media types according to the Accept request header; the media type will be conveyed by the Content-Type response header.
  • application/json

Responses

200

An array of meetings Meetings

0

Empty array of meetings

post /meeting
Send the invitation to a given meeting ID. (meetingPost)
Send invitations of the meeting to the participants by email.

Consumes

This API call consumes the following media types via the Content-Type request header:
  • application/x-www-form-urlencoded

Query parameters

meetingID (required)
Query Parameter — Unique identifier representing a specific meeting.

Produces

This API call produces the following media types according to the Accept request header; the media type will be conveyed by the Content-Type response header.
  • text/html; charset=utf-8

Responses

200

Success and returns detail of meeting in html text

Example data

Content-Type: text/html
<html><body>Show meeting details and guests</body></html>
